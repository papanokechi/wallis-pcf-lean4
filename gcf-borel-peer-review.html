2. Backward Recurrence for GCF Convergents

Cell 4

The generalized continued fraction $b_0 + \cfrac{a_1}{b_1 + \cfrac{a_2}{b_2 + \cdots}}$ is evaluated by backward recurrence: start with $t_N = 0$ and iterate $t_{n-1} = a_n / (b_n + t_n)$ for $n = N, N{-}1, \ldots, 1$. The convergent is $V = b_0 + t_0$ (or just $t_0$ if $b_0$ is absorbed).

Stability: backward recurrence is numerically stable when $|a_n/b_n| \to 0$ (convergent CF). For factorial-growth numerators it diverges classically, but the recurrence still produces finite truncations whose Borel limit can be compared to the analytic formula.

Python

gcf_limit() & gcf_convergent_sequence()

def gcf_limit(a_func, b_func, depth=200, b0=None):
    """Backward recurrence for GCF: b0 + K_{n>=1} a_n / b_n.
    
    Returns b0 + tail if b0 is given, otherwise just the tail t_0.
    """
    t = mp.mpf(0)
    for n in range(depth, 0, -1):
        a = mp.mpf(a_func(n))
        b = mp.mpf(b_func(n))
        t = a / (b + t)
    if b0 is not None:
        return mp.mpf(b0) + t
    return t


def gcf_convergent_sequence(a_func, b_func, depths, b0=None):
    """Compute convergents at each depth in the list. Returns list of (depth, value)."""
    return [(d, gcf_limit(a_func, b_func, depth=d, b0=b0)) for d in depths]

3. Lemma 1: Factorial Numerator with $k$-Shift Borel Regularization

Cell 6

GCF with $a_n = -n!$, $b_n = k$ (constant). Classically divergent by Stern–Stolz ($|a_n/b_n| = n!/k \to \infty$).

Borel regularization: the transform $\mathcal{B}_k(t) = k/(k+t)$ cancels factorial growth, and Laplace resummation gives

$V(k) = \displaystyle\int_0^\infty \frac{k\,e^{-t}}{k+t}\,dt = k\,e^k\,E_1(k)$

We first observe the divergent behavior of truncated convergents, then compute the analytic Borel value.

4. High-Precision Verification of Lemma 1

Cell 8

Three independent computations for each $V(k)$:

  1. Closed form: $k e^k E_1(k)$ via mpmath.e1
  2. Numeric Borel integral: $\int_0^\infty \frac{k e^{-t}}{k+t}\,dt$ via mpmath.quad
  3. Stieltjes transform: $k \int_0^\infty \frac{e^{-ks}}{1+s}\,ds$ (substitution $t = ks$)

All three must agree to 120+ digits.

5. Quadratic vs Linear Denominator GCF Computation

Cell 10

Two GCFs with $a(n) = 1$, both sharing $b(0) = 1$:

  • Quadratic: $b(n) = 3n^2 + n + 1$ → limit $V_\text{quad} \approx 1.19737\ldots$

6. High-Precision Target Values and Bessel Ratio Candidate Check

Cell 12

At 120-digit precision, compute $V_\text{quad}$ and $V_\text{lin}$ (both as full GCFs with $b_0$) and compare both to the Bessel candidate via the Perron–Pincherle identity for linear $b_n = \alpha n + \beta$:

$\text{GCF}(1,\,\alpha n + \beta) = \frac{I_{\beta/\alpha - 1}(2/\alpha)}{I_{\beta/\alpha}(2/\alpha)}$

For $b(n) = 3n+1$: $\alpha=3,\;\beta=1,\;\nu = 1/3$, so the candidate is $I_{-2/3}(2/3)\,/\,I_{1/3}(2/3) = 1 + I_{4/3}(2/3)\,/\,I_{1/3}(2/3) \approx 1.24150\ldots$ (using the recurrence $I_{\nu-1} = I_{\nu+1} + \tfrac{2\nu}{z}\,I_\nu$).

Expected result: The linear CF matches the Bessel ratio; the quadratic CF does not.

8. Ghost Identity Diagnosis: Parameter Sweep

Cell 16

Sweep over denominator functions with $a(n)=1$ and compare limits to Bessel/Airy candidates. For linear $b_n = \alpha n + \beta$, the Perron–Pincherle formula gives $\text{GCF}(1, \alpha n + \beta) = I_{\beta/\alpha-1}(2/\alpha)/I_{\beta/\alpha}(2/\alpha)$. This shows which parameterizations yield known closed forms and which do not.

# Perron formula for linear b_n = αn+β:
#   GCF(1, αn+β) = I_{β/α - 1}(2/α) / I_{β/α}(2/α)
def perron_bessel(alpha, beta):

10. Three-Term Recurrence Derivation for Quadratic Denominators

Cell 20

The convergents $P_n/Q_n$ satisfy the forward recurrence:

$P_n = b(n) \cdot P_{n-1} + a(n) \cdot P_{n-2}, \quad Q_n = b(n) \cdot Q_{n-1} + a(n) \cdot Q_{n-2}$

With $a(n)=1$ and $b(n) = 3n^2+n+1$:

$Q_n = (3n^2+n+1)\,Q_{n-1} + Q_{n-2}$

We verify forward vs backward consistency and analyze the dominant-balance growth rate of $Q_n$. The empirical error decay is $\log_{10}|e_n| \approx -0.41\,n^{3/2}$ (faster than Pincherle’s $-n\log n$ rate, which applies to linear $b_n$).

def forward_recurrence(a_func, b_func, N):
    """Forward recurrence for P_n, Q_n of CF b0 + K a_n/b_n.
    Returns list of (n, P_n, Q_n, P_n/Q_n)."""
    # P_{-1}=1, P_0=b_0; Q_{-1}=0, Q_0=1
    b0 = b_func(0)
    P_prev, P_curr = mp.mpf(1), mp.mpf(b0)
    Q_prev, Q_curr = mp.mpf(0), mp.mpf(1)
    results = [(0, P_curr, Q_curr, P_curr / Q_curr if Q_curr != 0 else mp.inf)]
    
    for n in range(1, N + 1):
        a = mp.mpf(a_func(n))
        b = mp.mpf(b_func(n))
        P_new = b * P_curr + a * P_prev
        Q_new = b * Q_curr + a * Q_prev
        P_prev, P_curr = P_curr, P_new
        Q_prev, Q_curr = Q_curr, Q_new
        if Q_curr != 0:
            results.append((n, P_curr, Q_curr, P_curr / Q_curr))
    return results

The three families of GCFs are unified by a single principle: the asymptotic growth of $b_n$ determines the analytic kernel, which determines the special-function family of the limit.

15. General Ghost-Identity Detection Criterion

Cell 30–31

A ghost identity occurs when a CF discovery pipeline confuses the output of one growth class with another (e.g., $1.197$ vs $1.241$).

Method: Fit $\log_{10}(Q_n) = c \cdot n \cdot \log_{10}(n) + d \cdot n + e$. The coefficient $c$ directly gives the polynomial degree of $b_n$:

  • $c \approx 1$ → linear $b_n$ → Bessel candidate
  • $c \approx 2$ → quadratic $b_n$ → NOT Bessel
  • $c \approx 3$ → cubic $b_n$ → NOT Bessel

# GHOST-IDENTITY DETECTOR — Three-test criterion
# ═══════════════════════════════════════════════════════════
mp.mp.dps = 80

def ghost_identity_detector(a_func, b_func, label="CF", ref_depth=300):
    # Three-test ghost-identity detector for a GCF.
    # Uses Q_n growth analysis: log₁₀(Q_n) ≈ c·n·log₁₀(n) + d·n + e
    # The coefficient c directly gives the polynomial degree of b_n.
    b0 = b_func(0)
    ref = gcf_limit(a_func, b_func, depth=ref_depth, b0=b0)
    
    P_prev, P_curr = mp.mpf(1), mp.mpf(b0)
    Q_prev, Q_curr = mp.mpf(0), mp.mpf(1)
    
    log_Q = []
    errors = []
    
    for n in range(1, 61):
        a = mp.mpf(a_func(n))
        b = mp.mpf(b_func(n))
        P_prev, P_curr = P_curr, b * P_curr + a * P_prev
        Q_prev, Q_curr = Q_curr, b * Q_curr + a * Q_prev
        
        if Q_curr != 0 and abs(Q_curr) > 1:
            lQ = float(mp.log10(abs(Q_curr)))
            log_Q.append((n, lQ))
            
            ratio = P_curr / Q_curr
            err = abs(ratio - ref)
            if err > 0:
                le = float(-mp.log10(err))
                errors.append((n, le))
    
    # PRIMARY DISCRIMINANT: Q_n growth coefficient
    # Fit: log₁₀(Q_n) = c·n·log₁₀(n) + d·n + e
    # c ≈ 1 for linear b_n, c ≈ 2 for quadratic, c ≈ 3 for cubic
    ns = np.array([d[0] for d in log_Q if d[0] >= 5], dtype=float)
    lQs = np.array([d[1] for d in log_Q if d[0] >= 5])
    
    A = np.column_stack([ns * np.log10(ns), ns, np.ones_like(ns)])
    coeffs = np.linalg.lstsq(A, lQs, rcond=None)[0]
    growth_degree = coeffs[0]
    
    # SECONDARY: Convergence acceleration ratio (late vs early digit gains)
    if len(errors) >= 15:
        early_rate = np.mean([d/n for n, d in errors if 3 <= n <= 8])
        late_rate = np.mean([d/n for n, d in errors if 15 <= n <= 25])
        accel_ratio = late_rate / early_rate if early_rate > 0 else 1
    else:
        accel_ratio = 1.0
    
    # CLASSIFY
    if growth_degree < 1.4:
        growth_class = "linear"
        is_bessel = True
    elif growth_degree < 2.5:
        growth_class = "quadratic"
        is_bessel = False
    else:
        growth_class = "cubic+"
        is_bessel = False
    
    return growth_class, is_bessel, {
        'growth_degree': round(growth_degree, 3),
        'accel_ratio': round(accel_ratio, 3)
    }

17. Forward/Backward Recurrence Stability Analysis

Cell 34–35

Backward recurrence ($t_{n-1} = a_n/(b_n + t_n)$) is contractive: perturbations in $t_N$ are attenuated by $\prod |a_n/b_n^2| \to 0$ exponentially.

Forward recurrence computes $P_n/Q_n$ as a ratio of exponentially growing quantities, losing precision through catastrophic cancellation.

Quantitative bound: For $b_n \geq Cn^\alpha$ with $\alpha \geq 1$:

$\text{backward error} \leq \prod_{k=n+1}^{N} \frac{1}{b_k^2} \approx \exp\!\bigl(-2\sum \alpha \log k\bigr)$

Theorem (Growth Coefficient). Let $Q_n = b_n Q_{n-1} + Q_{n-2}$ with $b_n \sim \alpha n^d$ for $\alpha > 0$, $d \geq 1$. Then:

$\log Q_n = \sum_{k=1}^n \log b_k + O(n) = d \cdot n \log n - (d - \log\alpha) \cdot n + O(\log n)$

Proof: Since $b_n \to \infty$, $Q_{n-2}/Q_{n-1} \to 0$, so $Q_n \sim b_n Q_{n-1}$. Telescoping: $\log Q_n \approx \sum_k \log(\alpha k^d) = d \sum \log k + n \log \alpha$. By Stirling, $\sum \log k = n \log n - n + O(\log n)$.

Corollary: Fitting $\log_{10}(Q_n) = c \cdot n \log_{10}(n) + dn + e$ extracts $c \to d = \deg(b_n)$. The ghost-identity detector (§15) is an asymptotic theorem, not a heuristic. $\square$
