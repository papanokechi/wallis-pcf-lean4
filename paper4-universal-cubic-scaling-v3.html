Abstract

We present a multi-agent AI discovery framework augmented with six frontier methodological innovations — renormalization group (RG) coarse-graining, interventional causal testing, automatic cross-domain isomorphism, Bayesian optimal experiment design, meta-law induction, and emergent agent feedback loops — applied to the discovery and validation of scaling laws across three physical domains. Our central finding is the concept of RG–Causal Convergence: scaling laws that are simultaneously fixed points of an RG flow (scale-invariant) and pass interventional causal tests. We identify two such convergent laws:

Materials: \(E_g \sim r_X^{-2/3}\)  (RG drift = 0.000, causal gap = 1.14)
Critical transitions: \(y \sim 0.150 \cdot x^3\)  (RG drift = 0.000, causal gap = 2.21)

The exoplanet domain reveals an RG flow phenomenon: the dominant exponent shifts from \(\beta = 2\) at full resolution to \(\beta = 3\) at coarser scales (4/5 scales), identifying the cubic law as an infrared fixed point with quadratic corrections as a relevant perturbation. Automatic isomorphism discovery maps \(\Delta_{\text{Hill}} \leftrightarrow \text{control\_param}\) (similarity = 0.781) without hand-coding. Among 35 laws tested, 25 survive Benjamini-Hochberg correction and all 9 unique causal tests pass. We report honest negative results: 0/30 automatic transfers succeed (scale mismatch), and meta-laws remain weak (R² = 0.07 from 35 laws). Statistical framework: BH FDR correction at \(\alpha = 0.05\).

2. Methods: Six Frontier Innovations

2.1 Innovation 1: Renormalization Group Coarse-Graining

Core Idea

If a scaling law \(y \sim x^\beta\) is truly universal, then the exponent \(\beta\) must be invariant under coarse-graining — i.e., it should not change when we systematically reduce the resolution of the data. This is the defining property of a fixed point in the renormalization group (RG). We import this concept from statistical physics where it is the standard tool for establishing universality.

Algorithm:

  1. Given dataset \(\{x_i, y_i\}_{i=1}^N\), define 5 resolution scales: \(\sigma \in \{1.0, 0.7, 0.5, 0.3, 0.2\}\)
  2. At each scale \(\sigma\), retain a random subset of \(\lfloor \sigma N \rfloor\) data points
  3. Run symbolic regression independently at each scale to discover the best single-term law \(y \sim \alpha \cdot x^\beta\)
  4. Extract the effective exponent \(\beta(\sigma)\) at each scale, accounting for composite features:
    • \(\texttt{control\_cube}^p \Rightarrow \beta_{\text{eff}} = 3p\)
    • \(\texttt{control\_sq}^p \Rightarrow \beta_{\text{eff}} = 2p\)
  5. Compute exponent drift: \(\text{drift} = \max_\sigma \beta(\sigma) - \min_\sigma \beta(\sigma)\)
  6. Fixed-point criterion: drift < 0.3 → RG fixed point (★); drift ≥ 0.3 → RG flow (✗)

Definition (RG Fixed Point).

A scaling law \(y = \alpha \cdot x^\beta\) is an RG fixed point if the recovered exponent \(\beta(\sigma)\) is invariant across all coarse-graining scales: \[ \text{drift}(\beta) \equiv \max_\sigma \beta(\sigma) - \min_\sigma \beta(\sigma) < 0.3 \] When drift = 0.000, the law is a perfect fixed point — the strongest possible evidence for scale invariance.

2.2 Innovation 2: Interventional Causal Testing

Core Idea

Symbolic regression discovers correlations. But a law \(y = f(x)\) is only physically meaningful if \(x\) causally drives \(y\). We test this using an interventional (do-calculus inspired) approach: if reshuffling \(x\) while preserving its marginal distribution destroys the law's predictive power, then \(x\) is genuinely causal.

Algorithm:

  1. For each discovered law \(y = f(x_1, \ldots, x_k)\), identify the primary variable \(x_j\) (highest absolute correlation with target)
  2. Compute observed R²: evaluate the law on the original data
  3. Perform intervention: randomly permute \(x_j\) while keeping all other variables and \(y\) fixed
  4. Compute interventional R²: evaluate the same law on the intervened data
  5. Compute causal gap: \(\Delta_{\text{causal}} = R^2_{\text{obs}} - R^2_{\text{int}}\)
  6. Causal criterion: \(\Delta_{\text{causal}} > 0.5\) and \(R^2_{\text{obs}} > 0.3\) → ★ CAUSAL

The logic: if \(x_j\) genuinely causes \(y\), then destroying the \(x_j \to y\) relationship by reshuffling \(x_j\) must collapse the fit. A large gap means the variable was doing real predictive work; a small gap would indicate spurious correlation or confounding.

2.3 Innovation 3: Automatic Domain Isomorphism

Core Idea

Previous versions used hand-coded variable mapping tables for cross-domain transfer. We replace this with automatic isomorphism discovery via statistical fingerprinting: each variable in each domain gets a 7-dimensional fingerprint, and variables are matched across domains by fingerprint similarity.

Fingerprint vector:

\[ \vec{f}(x) = \bigl(\mu(x),\ \sigma(x),\ \text{skew}(x),\ \text{kurt}(x),\ H(x),\ \rho(x,y),\ \hat{\beta}(x)\bigr) \]

where \(\mu\) = mean, \(\sigma\) = standard deviation, \(\text{skew}\) = Fisher skewness, \(\text{kurt}\) = excess kurtosis, \(H\) = entropy (via histogram), \(\rho(x,y)\) = Pearson correlation with target, and \(\hat{\beta}(x)\) = best-fit power-law exponent.

Matching: Weighted similarity with emphasis on the most physically meaningful features:

\[ \text{sim}(x_A, x_B) = 0.30 \cdot s_\rho + 0.30 \cdot s_\beta + 0.15 \cdot s_{\text{skew}} + 0.15 \cdot s_H + 0.10 \cdot s_{\text{kurt}} \]

Greedy assignment ensures each source variable maps to exactly one target variable (no repeats).

2.4 Innovation 4: Bayesian Optimal Experiment Design

Core Idea

Given two competing laws \(f_a\) and \(f_b\), find the next experiment (data point) that would maximally discriminate between them. This converts passive discovery into active science.

Algorithm:

  1. Sample 1000 candidate points from each feature's observed range
  2. Evaluate both competing laws at each candidate point
  3. Select the point \(x^*\) maximizing relative discrimination: \[ x^* = \arg\max_x \frac{|f_a(x) - f_b(x)|}{\max(|f_a(x)|, |f_b(x)|)} \]
  4. Report the driving feature (via systematic perturbation) and the discrimination power \(\in [0, 1]\)

2.5 Innovation 5: Meta-Law Induction

Core Idea

Discover laws about the laws themselves: which structural properties of a symbolic expression predict its quality (R²)? This is a "grammar of physics" — pattern extraction from the discovery process itself.

We extract 11 meta-features from each discovered law: number of terms, number of distinct variables, max/min/mean exponent, presence of negative exponents, maximum coefficient magnitude, complexity, single-term flag, integer exponent flag, and rational exponent flag. We then run symbolic regression on the meta-dataset \(\{(\vec{m}_i, R^2_i)\}_{i=1}^{35}\).

2.6 Innovation 6: Emergent Agent Feedback Loops

Core Idea

Address the v2 ablation finding that agents were a pipeline, not a swarm, by introducing feedback loops: Adversary failures modify Explorer search parameters; successful transferable exponents expand the grid; Meta-Learner insights adjust complexity penalties.

Three adaptive rounds with cross-round feedback:

  • Overfitting detected → reduce max terms from 3 to 2
  • Sensitivity failure → narrow exponent grid to integers only
  • Successful transfer exponent → expand grid to include ±0.5 neighborhood around that exponent

Methodological Note

The interventional test can detect causal relationships but cannot rule out unobserved confounders. In a true experimental setting, one would physically intervene on the system (e.g., modify planet masses). Our test is a computational analogue that uses statistical intervention (random permutation). For full causal identification, a structural causal model or a randomized experiment would be needed.

9. Honest Negatives & Limitations

Negative 1: Auto-Transfer Still Fails (0/30)

Despite the improved auto-isomorphism mapping, zero out of 30 automatic transfers succeeded. All transferred R² values are negative (range: −236,416 to −0.69). The auto-isomorphism correctly identifies which variables correspond across domains (e.g., \(\Delta_{\text{Hill}} \leftrightarrow \text{control\_param}\)), but the transferred expressions fail because of scale mismatch: the coefficient α from the source expression produces values on a completely different scale than the target.

Required fix (not yet implemented): z-score normalization of both source and target variables before transfer, and automatic coefficient refit on the target domain.

Negative 2: Meta-Laws Are Uninformative

The meta-law module discovered only weak patterns (R² < 0.08): higher exponents weakly correlate with higher R². With only 35 laws in the pool, the meta-dataset is too small for meaningful induction. This module needs 200+ laws across many domains and runs to produce useful results.

Negative 3: Adaptive Feedback Had No Observable Effect

The emergent feedback loops (Innovation 6) operated over 3 adaptive rounds, but the best laws in each domain were discovered in Round 1 and never improved. Round-over-round R² was flat: exoplanet (0.719 → 0.719 → 0.719), materials (0.532 → 0.532 → 0.532), critical transitions (0.999 → 0.999 → 0.999). The SR engine converges in one round; there is no room for incremental improvement with feedback.

Caveat: Synthetic Ground Truth Bias

The strongest RG–Causal Convergent result (critical transitions, drift = 0.000, gap = 2.21) uses synthetic data with a built-in cubic ground truth (\(y = 0.15x^3 - 0.3f + \epsilon\)). Recovering this exactly is expected, not surprising. The more scientifically interesting findings are: (a) the materials fixed point at β = −2/3, which is not built into the data generator, and (b) the exoplanet RG flow, which reveals genuine scale-dependent behavior.

Caveat: Data Quality Asymmetry

The three domains have fundamentally different data quality: critical transitions has a known analytic ground truth (N = 240, continuous target), materials uses bootstrap augmentation of 40 unique compositions (N = 160, continuous), and exoplanets use N-body simulations with binary labels converted to stability probabilities (N = 240). Cross-domain comparisons must account for these asymmetries.

10. Upgraded Predictions

Prediction 1: RG Fixed-Point Universality.

For any physical system exhibiting a fold catastrophe (A₃ singularity), the RG coarse-graining procedure will yield a fixed-point exponent \(\beta = 3.0 \pm 0.3\). Specifically: the exponent drift across 5 resolution scales should be < 0.3.

Prediction 2: RG Flow Pattern.

For systems where the cubic law is the correct asymptotic description but quadratic corrections are significant, the exponent should show a monotonic flow from β ≈ 2 at high resolution to β ≈ 3 at low resolution — matching the exoplanet pattern. This predicts that real Kepler multi-planet systems will show the same β : 2 → 3 flow under coarse-graining.

Prediction 3: Causal Gap Scaling.

The causal gap should scale with the law's true R²: for laws with R² > 0.9, the gap should exceed 2.0; for laws with R² ∈ [0.5, 0.9], gaps should be in [1.0, 2.0]. This follows from information-theoretic bounds on mutual information between the driving variable and the target.

Prediction 4: Auto-Isomorphism Stability.

The mapping \(\Delta_{\text{Hill}} \leftrightarrow \text{control\_param}\) should be recovered with similarity > 0.7 across different random seeds, dataset sizes, and noise realizations, confirming it reflects genuine structural correspondence rather than statistical coincidence.

Prediction 5 (Falsification Target).

For Ising model magnetization near \(T_c\), the RG analysis should yield a fixed-point exponent \(\beta \neq 3\) (mean-field predicts β = 1/2), and the cubic law \(y \sim x^3\) should not be an RG fixed point. This would confirm that the cubic class is restricted to fold catastrophes, not all critical phenomena.
