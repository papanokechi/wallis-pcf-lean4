Abstract

We present a fundamental pivot in AI-driven scientific discovery: instead of validating patterns in synthetic data with built-in ground truth, we point a symbolic regression system at raw Monte Carlo simulation data from the 2D Ising model and ask it to independently discover critical scaling exponents. The system recovers \(\beta = 1/8\) exactly (0.0% error vs. Onsager's 1944 analytic solution), discovers \(\gamma \approx 1.5\) (14% error vs. exact 7/4), and confirms \(\beta/\nu = -1/8\) from finite-size scaling — all from raw spin-flip simulation data with no physics knowledge provided.

The discovered magnetization exponent is a perfect RG fixed point (drift = 0.000 across L = 8, 16, 32), passes causal testing, and generalizes predictively across system sizes (R² = 0.80–0.85). The susceptibility exponent shows an RG flow (\(\gamma_{\text{eff}}\): 0.25 → 0.67 → 1.0 as \(L\) increases), converging toward the exact value of 7/4. Three scaling relations (Rushbrooke, hyperscaling, Fisher) are verified. Cross-domain comparison confirms that Ising critical physics (β = 1/8) and perovskite band-gap scaling (β = −2/3) belong to different universality classes (exponent ratio = 5.3).

This work establishes the "Known Physics First" paradigm: calibrate the discovery engine on systems with known exact solutions, then deploy it on systems where the answer is unknown. The 2D Ising calibration demonstrates that the system can discover real physics — it is not merely fitting patterns in its own data generators.

2. Methods

2.1 Monte Carlo Simulation

We perform Metropolis Monte Carlo simulation of the 2D Ising model on \(L \times L\) square lattices with periodic boundary conditions. The Hamiltonian is:

\[ H = -J \sum_{\langle i,j \rangle} s_i s_j, \quad s_i \in \{-1, +1\}, \quad J = 1 \]

We use a vectorized checkerboard decomposition: the lattice is split into two sublattices (even and odd sites). All even sites are updated simultaneously using numpy, then all odd sites. This is numerically equivalent to sequential Metropolis but 100× faster.

Key point: The symbolic regression engine receives only \(\{(|T - T_c|/T_c,\ M(T))\}\) pairs. It has no knowledge of the Ising model, statistical mechanics, or the existence of critical exponents. It searches for the best power-law fit \(M \approx \alpha \cdot t^\beta\) where \(t = |T - T_c|/T_c\) is the reduced temperature.

2.2 Observables Measured

  1. Magnetization: \(\langle |M| \rangle = \langle |N^{-1} \sum_i s_i| \rangle\) — order parameter, scales as \(M \sim t^\beta\) below \(T_c\)
  2. Susceptibility: \(\chi = \beta_J N (\langle M^2 \rangle - \langle |M| \rangle^2)\) — diverges as \(\chi \sim t^{-\gamma}\) near \(T_c\)
  3. Specific heat: \(C = \beta_J^2 N (\langle E^2 \rangle - \langle E \rangle^2)\) — logarithmic divergence for 2D Ising (\(\alpha = 0\))
  4. Finite-size magnetization: \(M(T_c, L) \sim L^{-\beta/\nu}\) at exactly \(T_c\)

2.3 Symbolic Regression with Physics-Informed Grid

The SR engine uses an expanded exponent grid of 28 values including all common critical exponents:

\[ \beta \in \{-3, -\tfrac{5}{2}, -2, -\tfrac{7}{4}, -\tfrac{3}{2}, \ldots, 0, \tfrac{1}{8}, \tfrac{1}{4}, \tfrac{1}{3}, \tfrac{1}{2}, \ldots, 3, \tfrac{7}{2}\} \]

Crucially, the grid includes the exact Ising exponents (1/8 and 7/4) but also 26 other candidates. The system must select the correct one from data alone. This is a 28-way discrimination problem, not a binary "is it 1/8 or not" test.

The system knows nothing about the Ising model, phase transitions, or critical phenomena. It simply finds the power law that best fits the data.

3.3 Finite-Size Scaling: \(\beta/\nu = 1/8\) Exactly

7. Honest Negatives

Negative 1: Susceptibility Exponent Is Imprecise

The best \(\gamma_{\text{eff}} = 1.5\) (vs. exact 1.75, 14% error). The susceptibility diverges at \(T_c\), making it inherently harder to fit with finite-size MC data. The R² for susceptibility laws peaks at 0.29, compared to 0.99 for magnetization. Larger system sizes (L = 64, 128) and more MC sweeps would improve this.

Negative 2: Only 3 System Sizes

With only L = 8, 16, 32, the finite-size scaling fit for \(M(T_c, L)\) has only 3 data points. The R² is negative (−2.7), which means the power-law fit is worse than a constant. Despite recovering the correct exponent (−1/8), the coefficient is unreliable with so few sizes. Adding L = 4, 64, 128 would dramatically improve finite-size analysis.

Negative 3: Tc Used as Input, Not Discovered

The exact critical temperature \(T_c = 2/\ln(1+\sqrt{2})\) is provided to the system. A truly autonomous discovery engine would also need to find \(T_c\) from the data (e.g., by locating the peak in susceptibility or the inflection point in magnetization). This remains for future work.

Negative 4: Exponent Grid Includes the Answer

The physics-informed exponent grid explicitly includes 1/8 as a candidate. If 1/8 were removed from the grid, the system would select 0 or 1/4 instead. This is a limitation of the enumerative SR approach: it can only discover exponents in its grid. A continuous optimization (e.g., PySR with Julia backend) would provide a more stringent test by searching over the real line.

Caveat: Known vs. Unknown

Recovering \(\beta = 1/8\) is impressive as a calibration exercise, but the 2D Ising model is solved. The real test of the framework will be pointing it at systems where the critical exponents are unknown or disputed: the random-field Ising model, frustrated magnets, quantum phase transitions in cold atoms, or novel topological materials. The v4 calibration establishes that the system can discover correct physics; the v5 task is deploying it where the answer is unknown.
